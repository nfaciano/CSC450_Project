Detailed Description

Abstract base class for a one-dimensional mathematical function.

This class provides a common interface for functions of a single variable. It supports evaluation of the function at a point, checking if a point is within the function's domain, and obtaining the function's domain bounds. It is designed to be extended by more specific function types.
